# Resetting Commits

## Undo the Latest Commit

### `git reset HEAD~1`

Removes the latest commit while keeping all changes unstaged.

```bash
$ git reset HEAD~1

Unstaged changes after reset:
M   app.py
```

---

## Unstage a File

### `git reset HEAD file.txt`

Removes a file from the staging area while keeping its changes.

```bash
$ git reset HEAD app.py

Unstaged changes after reset:
M   app.py
```

---

## Undo the Latest Commit and Keep Changes Staged

### `git reset --soft HEAD~1`

Removes the latest commit while keeping all changes staged.

```bash
$ git reset --soft HEAD~1

$ git status

Changes to be committed:
    modified: app.py
```

---

## Undo the Latest Commit and Delete Changes

### `git reset --hard HEAD~1`

Removes the latest commit and permanently discards all changes.

```bash
$ git reset --hard HEAD~1

HEAD is now at 8f3e2c1 Added login feature
```
