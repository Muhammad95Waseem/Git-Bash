# Basic Bash Commands for Git

These are common Bash commands used while working with Git repositories.

---

## Print Current Directory

### `pwd`

Displays the full path of the current working directory.

```bash
$ pwd
/c/Users/Waseem/projects
```

---

## List Files and Folders

### `ls`

Shows files and folders in the current directory.

```bash
$ ls
README.md  app.py  docs/
```

### `ls -a`

Shows all files, including hidden files like `.git`.

```bash
$ ls -a
.  ..  .git  README.md  app.py
```

---

## Change Directory

### `cd folder-name`

Moves into a specified directory.

```bash
$ cd my-project
```

### `cd ..`

Moves to the parent directory.

```bash
$ cd ..
```

### `cd`

Moves to the home directory.

```bash
$ cd
```

---

## Create Files and Folders

### `mkdir folder-name`

Creates a new directory.

```bash
$ mkdir docs
```

### `touch file-name.extension`

Creates an empty file.

```bash
$ touch README.md
```

---

## Remove Files and Folders

### `rm file-name.extension`

Deletes a file.

```bash
$ rm temp.txt
```

### `rm -r folder-name`

Deletes a directory and its contents.

```bash
$ rm -r docs
```

---

## Rename or Move Files

### `mv old-name new-name`

Renames or moves a file or directory.

```bash
$ mv notes.txt README.md
```

---

## Copy Files

### `cp source destination`

Copies a file.

```bash
$ cp README.md backup.md
```

---

## Display File Contents

### `cat file-name.extension`

Prints the contents of a file.

```bash
$ cat README.md
# My Project
```

---

## Clear Terminal Screen

### `clear`

Clears all previous terminal output.

```bash
$ clear
```
