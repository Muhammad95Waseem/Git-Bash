# Branches

## Viewing Branches

### `git branch`

Lists local branches.

```bash
$ git branch

* main
  feature-login
  bug-fix
```

---

### `git branch -a`

Lists local and remote branches.

```bash
$ git branch -a

* main
  feature-login
  remotes/origin/main
  remotes/origin/dev
```

---

## Creating a Branch

### `git branch branch-name`

Creates a new branch without switching to it.

```bash
$ git branch feature-login
```

---

### `git switch -c branch-name`

Creates a new branch and switches to it.

```bash
$ git switch -c feature-login

Switched to a new branch 'feature-login'
```

---

## Switching Branches

### `git switch branch-name`

Switches to an existing branch.

```bash
$ git switch main

Switched to branch 'main'
```

---

## Comparing Branches

### `git diff branch1 branch2`

Shows differences between two branches.

```bash
$ git diff main feature-login
```

---

## Merging Branches

### `git merge branch-name`

Merges the specified branch into the current branch.

```bash
$ git merge feature-login

Updating 8f3e2c1..a3f5c1d
Fast-forward
 app.py | 5 +++++
```

---

### `git checkout --ours index.html`

Keeps the current branch's version of a conflicted file.

```bash
$ git checkout --ours index.html
```

---

### `git checkout --theirs index.html`

Keeps the incoming branch's version of a conflicted file.

```bash
$ git checkout --theirs index.html
```

---

## Renaming a Branch

### `git branch -m old-name new-name`

Renames a branch.

```bash
$ git branch -m feature-login login-feature
```

---

## Deleting a Branch

### `git branch -d branch-name`

Deletes a branch that has already been merged.

```bash
$ git branch -d feature-login

Deleted branch feature-login (was a3f5c1d).
```

---

### `git branch -D branch-name`

Force deletes a branch, even if it has unmerged changes.

```bash
$ git branch -D feature-login

Deleted branch feature-login (was a3f5c1d).
```
