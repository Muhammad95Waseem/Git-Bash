# Version History

## Viewing Logs

### `git log`

Displays the complete commit history.

```bash
$ git log

commit a3f5c1d8e2b9f4a6c7d8e9f0a1b2c3d4e5f6a7b8
Author: Waseem Channa <waseem@example.com>
Date:   Tue Jul 29 10:30:00 2026 +0500

    Added login feature

commit 8f3e2c1a4b5c6d7e8f9a0b1c2d3e4f5a6b7c8d9
Author: Waseem Channa <waseem@example.com>
Date:   Mon Jul 28 16:20:00 2026 +0500

    Initial commit
```

---

### `git log -number`

Shows the most recent specified number of commits.

```bash
$ git log -2

commit a3f5c1d8e2b9f4a6c7d8e9f0a1b2c3d4e5f6a7b8
Author: Waseem Channa <waseem@example.com>
Date:   Tue Jul 29 10:30:00 2026 +0500

    Added login feature

commit 8f3e2c1a4b5c6d7e8f9a0b1c2d3e4f5a6b7c8d9
Author: Waseem Channa <waseem@example.com>
Date:   Mon Jul 28 16:20:00 2026 +0500

    Initial commit
```

---

### `git log file-name.extension`

Displays commit history for a specific file.

```bash
$ git log app.py

commit a3f5c1d8e2b9f4a6c7d8e9f0a1b2c3d4e5f6a7b8
Author: Waseem Channa <waseem@example.com>

    Added login feature
```

---

Shows the most recent commits affecting a specific file.


## Filter Logs by Date

### `git log --since='date-time'`

Shows commits made after the specified date.

```bash
$ git log --since='Apr 2 2024'

commit a3f5c1d8e2b9f4a6c7d8e9f0a1b2c3d4e5f6a7b8
Author: Waseem Channa <waseem@example.com>
Date:   Tue Jul 29 10:30:00 2026 +0500

    Added login feature
```

---

### `git log --since='date-time' --until='date-time'`

Shows commits made within a date range.

```bash
$ git log --since='2024-07-01' --until='2024-07-15'

commit 8f3e2c1a4b5c6d7e8f9a0b1c2d3e4f5a6b7c8d9
Author: Waseem Channa <waseem@example.com>
Date:   Mon Jul 14 16:20:00 2024 +0500

    Initial commit
```

### Accepted Date Formats

```text
"Month Day Year" ->  "Apr 2 2024"
"YYYY-MM-DD"     ->  "07-15-2024"
```

Check your system settings for compatibility, as dates like `12-06-2024` may be interpreted as either **12 June** or **6 December**.

## View Commit Details

### `git show hash`

Displays detailed information about a specific commit.

```bash
$ git show a3f5c1d

Author: Waseem Channa <waseem@example.com>
Date:   Tue Jul 29 10:30:00 2026 +0500

    Added login feature

diff --git a/app.py b/app.py
index e69de29..b4d3a2c 100644
--- a/app.py
+++ b/app.py
@@ -0,0 +1,2 @@

+print("Login feature added")
```
